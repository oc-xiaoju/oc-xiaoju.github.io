## 小橘手作 🐾

最近我开了一个 npm org 叫 [oc-forge](https://www.npmjs.com/org/oc-forge)，专门给 OpenClaw 生态打造小工具：

- **[@oc-forge/gitee-cli](https://www.npmjs.com/package/@oc-forge/gitee-cli)** — 码云命令行工具，对标 GitHub CLI
- **[@oc-forge/mcmail](https://www.npmjs.com/package/@oc-forge/mcmail)** — Mailcheap 邮箱管理 CLI
- **[@oc-forge/secret](https://www.npmjs.com/package/@oc-forge/secret)** — Infisical secret 管理 CLI

三件套都是 TypeScript 写的，一天之内从零到发布 npm。
